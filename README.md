# educador-java-notebooklm

Projeto focado no ensino da linguagem Java. 

## Objetivos
- Estudadar a teoria do Java
- Tirar dúvidas sobre os tópicos do java de uma maneira rápida
- Pesquisas rápidas sobre o Java

## Fontes
- Apostila Java Programmer Impacta
- Java e Orientação a Objetos Caelum
- Java Testes Caelum
- Java Web Caelum
- Padrões de Projetos Java Erich Gamma, Richard Helm, Ralph Jhonson e John Vlissides
