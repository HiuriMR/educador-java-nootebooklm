# 📚 educador-java-notebooklm

Este projeto foi criado com o objetivo de estruturar meu aprendizado em **Java** utilizando o NotebookLM como ferramenta de apoio para estudo, síntese e aprofundamento.

O foco principal foi transformar conteúdos clássicos da linguagem em um **sistema organizado de conhecimento**, permitindo não apenas aprender, mas também revisar e aplicar os conceitos de forma eficiente.

### Objetivos de estudo:

* Consolidar fundamentos sólidos em Java
* Dominar conceitos de Orientação a Objetos
* Entender testes automatizados na prática
* Aprender desenvolvimento Java Web
* Aplicar padrões de projeto utilizados no mercado
* Desenvolver autonomia no uso de IA como ferramenta de aprendizado

---

## 📖 Curadoria de Fontes

## 📘 Miniguia de Estudo (Entrega Final)

### 🧾 Resumo Estruturado

#### 🔹 Fundamentos de Java

* Linguagem orientada a objetos
* Tipagem forte
* Uso de classes e métodos
* Controle de fluxo (if, for, while)

#### 🔹 Orientação a Objetos

* Encapsulamento protege dados
* Herança promove reutilização
* Polimorfismo permite flexibilidade
* Interfaces definem contratos

#### 🔹 Testes em Java

* Testes unitários validam comportamento
* JUnit como principal framework
* TDD melhora qualidade do código

#### 🔹 Java Web

* Servlets processam requisições
* JSP renderiza páginas dinâmicas
* MVC organiza a aplicação

#### 🔹 Padrões de Projeto

* Singleton: uma única instância
* Factory: criação de objetos
* Strategy: troca de comportamento
* Observer: comunicação entre objetos

---

### 📚 Glossário

* **Encapsulamento**: proteção dos dados internos de uma classe
* **Herança**: reutilização de código entre classes
* **Polimorfismo**: múltiplas formas de comportamento
* **Interface**: contrato de implementação
* **TDD**: desenvolvimento orientado a testes
* **Servlet**: componente para aplicações web Java
* **Design Patterns**: soluções reutilizáveis para problemas comuns
